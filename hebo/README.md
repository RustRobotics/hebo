- [Documentation](https://docs.rs/hebo)
- [Release notes](https://github.com/RustRobotics/hebo/releases)

## Build on Linux
First install dependencies:
```bash
sudo apt install -y \
  gcc \
  libhiredis-dev \
  libmongoc-dev \
  libmariadb-dev \
  libpq-dev
```
